# **App Name**: NutriJoy

## Core Features:

- Swift Mobile-First Food Logging: A highly responsive interface for rapid and intuitive entry of food items, capturing calorie, sugar, and sodium data with minimal taps.
- Offline-First Data Storage: Utilizes a local database (e.g., IndexedDB) for all user-generated data, guaranteeing full offline functionality and persistent data access.
- Personalized Calorie & Nutrient Goals: Calculates daily calorie needs (TDEE-based), establishes personalized limits for sugar and sodium (WHO-based/configurable), and offers guidance for weight management goals.
- Activity Energy Burn Estimator: Estimates calories burned from various daily activities and work, providing a comprehensive overview of the user's total energy expenditure.
- Dynamic Daily Progress Summary: Interactive visual dashboards displaying real-time progress for calories consumed versus burned, and key nutrient intake against set goals, with clear warning indicators.
- Secure Local Data Management: Enables users to easily export and import their local nutritional data (e.g., JSON) for reliable backup, restoration, and data portability.
- AI-Enhanced Encouraging Feedback: A tool that generates personalized, friendly microcopy and positive affirmations based on tracking data to motivate users and promote healthy habits without a shaming tone.

## Style Guidelines:

- Primary color: A lively, warm red-orange (#E54A14) conveying energy and positivity, suitable for calls to action and key informational elements.
- Background color: A subtly warm off-white (#F9F4F3), creating a clean and spacious feel that ensures readability and reduces eye strain.
- Accent color: A vibrant pink (#F880A3), used for highlights, interactive elements, and adding a playful contrast to the primary palette.
- Headlines will use 'Poppins' (geometric sans-serif) for a modern, precise, and approachable feel, while body text will use 'PT Sans' (humanist sans-serif) for its readability and warmth, creating a friendly and clean aesthetic.
- Utilize modern, clean line-art or solid-fill icons with consistently rounded shapes, enhancing the app's playful yet sophisticated visual identity.
- A mobile-first, card-based layout featuring generous white space, clear visual hierarchy, and large, accessible tap targets for seamless navigation and minimal cognitive load.
- Incorporate subtle and swift animations for screen transitions, data updates, and interactive feedback, ensuring a smooth and responsive user experience without causing delay.
